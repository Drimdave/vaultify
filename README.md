# Vaultify

Vaultify is a premium mobile first fintech landing page built with Next.js and Tailwind CSS. The platform provides a secure and fee free financial ecosystem designed for modern users.

## Core Features

* High Fidelity Glassmorphism: Modern aesthetic with sleek translucent components and blurred backgrounds.
* Responsive Navigation: Interactive mobile hamburger menu and dynamic desktop navigation with active section highlighting.
* Smooth Scrolling: Seamless transitions between sections for an enhanced user experience.
* Trust Indicators: Visual social proof with high resolution user avatars and global currency statistics.
* Performance Optimized: Built on the Next.js App Router for superior loading speeds and SEO.

## Technology Stack

* Framework: Next.js
* Styling: Tailwind CSS
* Logic: TypeScript
* Components: React

## Development Setup

1. Install dependencies:
   npm install

2. Run the development server:
   npm run dev

3. Open http://localhost:3000 in your browser to view the application.

## Deployment

This project is optimized for deployment on Vercel. Connect your GitHub repository to Vercel for automatic deployments on every push.
